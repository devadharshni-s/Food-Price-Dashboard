# Food-Price-Dashboard
An interactive Power BI dashboard that visualize food prices across India, highlighting state-wise trends, commodity comparisions, and key market insights.

Tools Used
-Excel: Data cleaning, transformation, shaping
-Power BI: Modeling, visualization
-DAX: Custom measures for average price, volatility, category segments
-Maps & Charts: State-wise view, trend analysis, top commodities

Key Features
-KPI cards: top commodity price, volatility, markets covered
-Multi-year average price trend analysis
-State-level heat map
-Price distribution by food categories
-Top 10 expensive commodities
-Pie chart for market coverage by states

Files Included
-Cleaned dataset (CSV/XLSX)
-Power BI report (.pbix)
-Screenshots
-Documentation

How to Use
-Download the PBIX file
-Open in Power BI Desktop
-Interact with filters, time slicers, and visuals

Future Enhancements
-Add forecasting using DAX/Power BI AI visuals
-Automate data refresh using Python or Power BI service
-Add anomaly detection for price spikes
